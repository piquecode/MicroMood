# Changelog

All notable changes to MicroMood will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2025-01-17
### Added
- Weekly summary card showing current streak and mood frequency for the week
- Floating bottom navigation bar with persistent visibility while scrolling

### Changed
- Redesigned navigation bar with floating card design and minimal text labels
- Improved navigation accessibility with always-visible controls
- Enhanced visual hierarchy with elevated navigation component

### Improved
- Better dark mode support for AMOLED displays with refined contrast
- Consistent theming across light and dark modes for navigation elements

## [1.0.0] - 2025-01-11
### Added
- Simple mood tracking with 5 mood levels
- Add notes to mood entries
- Date picker for logging past moods
- Mood trends chart
- Calendar view with mood history
- CSV export/import for data backup
- Daily reminder notifications
- Dark mode support
- Privacy-focused local storage
- Initial release
