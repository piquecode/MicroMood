# Changelog

All notable changes to MicroMood will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-11

### Added
- Simple mood tracking with 5 mood levels
- Add notes to mood entries
- Date picker for logging past moods
- Mood trends chart
- Calendar view with mood history
- CSV export/import for data backup
- Daily reminder notifications
- Dark mode support
- Privacy-focused local storage
- Initial release
